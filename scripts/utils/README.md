# Scripts Utilitaires

Ce dossier contient les scripts pour gérer le dataset et l'hébergement sur Hugging Face.

## Scripts Disponibles

### `enrich_all_films.py`
Enrichit TOUS les films avec l'API TMDB.

### `upload_to_huggingface.py`
Upload le dataset sur Hugging Face.

### `clean_hf_cache.py`
Nettoie le cache Hugging Face.
